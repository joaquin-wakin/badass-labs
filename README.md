# Badass Labs &#8211; Nano Shield

## Run Locally

1. Download or clone the project

   ```sh
   git clone https://github.com/joaquin-wakin/badass-labs.git
   ```

2. Go to the project directory

   ```sh
   cd badass-labs
   ```

3. Install dependencies

   ```sh
   yarn install
   ```

4. Start the local server

   ```sh
   yarn dev
   ```
